# hashmap-benchmark

A hash map benchmark, include Java, CSharp, golang and so on.

## Install

Dependent on:

* `flat_hash_map`: https://github.com/skarupke/flat_hash_map
* `jstd_hashmap`: https://gitee.com/shines77/jstd_hashmap

Update submodule(s):

```shell
git submodule init
git submodule update --remote --recursive
```
